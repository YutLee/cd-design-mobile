# Divider 分割线
区隔内容的分割线。

## 使用场景

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。

## API
属性 | 说明 | 类型 | 默认值
---|---|---|---
children | 嵌套的标题 | ReactNode | -
className | 分割线样式类 | string | -
dashed | 是否虚线 | boolean | false
align | 分割线标题的位置 | `left` \| `right` \| `center` | `center`
type | 水平还是垂直类型 | `horizontal` \| `vertical` | `horizontal`
