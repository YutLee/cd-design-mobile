# Radio 单选按钮
单选按钮。

## 使用场景
- 用于在多个备选项中选中单个状态。
- 和 `Select` 的区别是，`Radio` 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

## API
属性 | 说明 | 类型 | 默认值
---|---|---|---
checked | 指定当前是否选中 | boolean | false
disabled | 是否禁用 | boolean | false
name | 名称 | string | -
value | 值 | value | -
onChange | 变化时回调函数 | function(event: Event) | -
